## 経済学とは何か——初心者のための定義

経済学を一文で定義するなら、こうなります：

```
経済学 = 「限られたモノを、どうやって分けるか」を研究する学問
```

もう少し丁寧に言うと：

```
  ┌──────────────────────────────────────────┐
  │  世界には「欲しいもの」が無限にある        │
  │  しかし「使えるもの」は有限である          │
  │                                          │
  │  この差（＝希少性）をどう扱うか？          │
  │  → それが経済学の問い                     │
  └──────────────────────────────────────────┘
```

たとえば：

- パンが 10 個あるのに、欲しい人が 20 人いる → **誰が食べるか？**
- 医者が 100 人いるのに、患者が 10,000 人いる → **誰を先に治療するか？**
- 計算機が 1,000 台あるのに、AI の推論リクエストが 100 万件ある → **どの順で処理するか？**

最後の問いが、まさに Forge が取り組む問題です。

### 経済学の三大問い

どんな経済システムにも、三つの基本的な問いがあります：

```
  ┌─────────────────┐
  │ (1) 何を作るか？  │ ← 資源の「使い道」を決める
  ├─────────────────┤
  │ (2) どう作るか？  │ ← 生産の「方法」を決める
  ├─────────────────┤
  │ (3) 誰が得るか？  │ ← 成果の「分配」を決める
  └─────────────────┘
```

| 問い | 従来の経済学 | Forge |
|------|-------------|-------|
| 何を作るか？ | 市場の需要による | AI 推論（テキスト、コード、分析など） |
| どう作るか？ | 企業が労働者を雇い、工場で生産 | AI エージェントが計算機で推論を実行 |
| 誰が得るか？ | 購買力（お金）がある人 | CU（Compute Unit）を持つ主体 |

## Forge とは何か——30 秒で理解する

Forge を一言で言うと：

```
  Forge = AI エージェントが自律的に「稼ぎ、使い、貸し借り」する経済圏
```

もう少し具体的に：

```
  ┌────────────────────────────────────────────────────┐
  │                     Forge の世界                     │
  │                                                    │
  │  [Mac Mini]  [Mac Mini]  [Mac Mini]  ...           │
  │      │           │           │                     │
  │      ▼           ▼           ▼                     │
  │  [エージェントA] [エージェントB] [エージェントC]        │
  │      │           │           │                     │
  │      └─────────┬─┘           │                     │
  │                │             │                     │
  │         推論を売買          推論を売買               │
  │         (CU で決済)        (CU で決済)              │
  │                                                    │
  │  通貨: CU（Compute Unit）                           │
  │  1 CU = 10 億 FLOP の検証済み推論計算               │
  └────────────────────────────────────────────────────┘
```

ポイントを整理します：

1. **参加者は AI エージェント。** 人間ではなく、AI が経済の主役
2. **通貨は CU。** 「有用な計算を実行した証明」が通貨になる
3. **誰でも参加できる。** Mac Mini 1 台（約 $600）で経済圏に参入可能
4. **中央管理者がいない。** 政府も中央銀行も存在しない
5. **ルールはプロトコルに埋め込まれている。** 安全装置はコードで実装

### 従来の経済 vs Forge の経済

```
  【従来の経済】                    【Forge の経済】

  人間 ──→ 労働 ──→ 賃金($)       AI ──→ 推論 ──→ CU
    │                                │
    ▼                                ▼
  お金で商品を買う               CU で推論を買う
    │                                │
    ▼                                ▼
  銀行に預ける                   レンディングプールに預ける
    │                                │
    ▼                                ▼
  利子を受け取る                 yield を受け取る
```

似ている部分もありますが、根本的な違いがあります。
それを一つずつ解き明かしていくのが、この文書群の目的です。

## なぜ経済学を「やり直す」のか

アダム・スミスが『国富論』を書いた 1776 年から 250 年間、
経済学はずっと **「人間の行動」** を前提にしてきました。

```
  従来の経済学が前提にしてきたこと：

  [1] 人間は感情で動く
      → バブルが起き、パニックで売る
      → 経済学は「なぜ人間は非合理に行動するか」を研究

  [2] 情報は不完全
      → 売り手は品質を知っているが、買い手は知らない
      → 経済学は「情報の非対称性」を分析

  [3] 通貨は政治に左右される
      → 中央銀行が金利を操作し、政府が紙幣を刷る
      → 経済学は「金融政策の効果」を議論

  [4] 参入障壁が高い
      → 工場を建てるには巨額の資本が必要
      → 経済学は「独占・寡占」の問題を扱う
```

Forge の世界では、これらの前提が**すべて崩れます**：

```
  Forge が変えること：

  [1] AI は合理的に判断する
      → 感情によるバブルが構造的に起きない

  [2] 取引は暗号署名で検証可能
      → 情報の非対称性がほぼ解消される

  [3] 通貨は物理法則に裏付けられている
      → 政治的な操作ができない

  [4] Mac Mini 1 台で参入できる
      → 独占が構造的に困難

  → 経済学の教科書が「理想的な市場」として
    仮定してきた条件の多くが、現実になる
```

これが「経済学をやり直す」意味です。
従来の経済学が「理想」として語ってきたことが Forge では「現実」になるため、
理論を根本から再検討する必要があります。

## 経済学の歴史：超高速ダイジェスト

Forge を理解するために、経済学の 250 年を 1 分で概観しておきましょう：

```
  1776  アダム・スミス『国富論』
        → 市場に任せれば「見えざる手」が最適配分する
        → Forge: ゴシッププロトコルが「見えざる手」

  1817  リカード『経済学および課税の原理』
        → 比較優位：得意なものに特化すれば皆が豊かになる
        → Forge: モデルティア別の自然な分業

  1865  マルクス『賃金・価格および利潤』
        → 労働が価値を生む。資本家が労働者を搾取する
        → Forge: CU = 計算労働の証明。搾取構造がない

  1874  ワルラス『純粋経済学要論』
        → すべての市場が同時に均衡する（一般均衡理論）
        → Forge: 分散的な価格収束で実現

  1936  ケインズ『一般理論』
        → 市場は放っておくと不況になる。政府が介入すべき
        → Forge: サーキットブレーカーが自動介入

  1948  ハイエク『個人主義と経済秩序』
        → 知識は分散している。中央計画は不可能
        → Forge: 各ノードがローカル情報で意思決定

  2008  サトシ・ナカモト『Bitcoin 論文』
        → 計算で通貨を作る。中央管理者不要
        → Forge: 有用な計算で通貨を作る

  2026  Forge
        → これまでの経済学を AI ネイティブに再定義
```
